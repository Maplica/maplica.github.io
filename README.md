# Maplica - home site

This project is a minimal Next.js page.

Quick start

```bash
npm install
npm run dev
```

Features
- Next.js + TypeScript
- Tailwind CSS
- Blog posts written in MDX in `content/posts`

Content
- Landing page at `/`
- Blog index at `/blog`
- Blog posts at `/blog/[slug]`
- Solution list at `/solutions`

Notes
- After cloning, run `npm install` to install dependencies.
- Add more posts to `content/posts` as `.md` files with frontmatter (title, date, description).
